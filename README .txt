
To be able to reserve tables, you need to connect the server.
To connect the server you need:
-Install node.js
-Enter the command line
-Go to: Project\server>node server.js

To display menus and events, you need to allow browsers to download local files using Ajax.
In order for the browsers could download local files using Ajax:
-Install node.js
-Install npm package globally npm -g install static-server
-Go to the folder "Project"
-Run the server static-server: write "static-server" to the command line and press Enter
-Follow the link: http://localhost:9080/index.html
-Or you can choose any other way to serve local files
